# 🚀 SkillMap - Career Roadmap Generator

SkillMap is an interactive web application that helps students explore different career paths and follow a structured learning roadmap. It provides progress tracking, achievement badges, certifications, project recommendations, and learning resources to support career growth.

## 📌 Features

✅ Career Roadmaps for multiple domains
✅ Progress Tracking System
✅ Achievement Badges
✅ Recommended Certifications
✅ Project Recommendations
✅ Learning Resources
✅ Career Tips & Guidance
✅ Download Progress Report
✅ Reset Progress Option
✅ Responsive and Modern UI

## 🎯 Supported Career Paths

- 👨‍💻 Web Developer
- ☕ Java Developer
- 🤖 AI / ML Engineer
- 📊 Data Analyst
- 🔒 Cyber Security Analyst

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
